안녕하세요, 웹 퍼블리셔 김서현입니다.

저는 사용자가 웹사이트를 통해 새롭고 즐거운 경험을 할 수 있도록 다양한 표현 기법을 연구하고, 이를 바탕으로 기억에 남는 웹사이트를 만들기 위해 노력합니다. 디자인을 단순히 구현하는 것을 넘어, 사용자 경험을 최우선으로 고려한 웹사이트를 만드는 것이 제 목표입니다. 기술과 창의력을 결합하여 더 나은 마크업을 제공하고, 지속적으로 성장하는 퍼블리셔가 되겠습니다.

SKILLS

HTML

HTML 기술을 기본적인 웹 페이지 구조 작성부터, 폼 처리, 시맨틱 태그 사용, 반응형 웹 디자인 구현까지 수행할 수 있습니다.

CSS

웹 페이지의 레이아웃, 색상, 글꼴 스타일링, 반응형 디자인 구현, 애니메이션 효과 등을 적용할 수 있습니다.

JavaScript

이미지 편집, 배너 및 웹 그래픽 디자인, 사진 보정, 레이어 및 마스크 활용 등 다양한 디자인 작업을 할 수 있습니다.

WordPress

다양한 유형의 웹사이트를 구축하고, 테마 및 플러그인 설정, 콘텐츠 관리, 사이트 최적화 등을 할 수 있습니다.

PROJECT

Project Title - EAT IT

홈카페 웹사이트, EAT IT은 홈카페 레시피를 제공하는 웹사이트로 다양한 커피 음료와 디저트를 집에서도 쉽게 만들 수 있도록 돕는 곳입니다. EAT IT은 홈카페를 즐기려는 사람들에게 유용한 정보를 제공하고, 커피와 디저트 만들기를 더욱 즐겁고 쉽게 만들어줍니다.

Project Title - 명지전문대학 정보통신공학과

명지전문대학교 정보통신공학과의 새 웹사이트는 학과 정보 전달과 학생들의 학문적, 진로적 지원을 강화하기 위해 만들어졌습니다. 이전 웹사이트의 부족한 부분을 개선하여, 교육 목표, 커리큘럼, 교수진 정보 등을 명확하게 제공하고 있습니다. 또한, 학생들이 쉽게 정보를 찾을 수 있도록 직관적인 디자인과 다양한 기능을 추가해 학사 생활을 지원하고 있습니다.

Project Title - KISETSU

KISETSU는 일본의 사계절을 대표하는 독창적이고 아름다운 디저트를 소개하는 플랫폼입니다. 일본의 전통과 현대적인 감각이 어우러진 디저트를 통해 계절의 변화를 느낄 수 있습니다. 일본 디저트를 사랑하는 분들은 물론, 일본 문화와 계절감에 흥미를 가진 모든 이들에게 영감을 제공하며 새로운 즐거움을 선사합니다.
